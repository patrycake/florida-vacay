Tic Tac Toe

Needs not explanation

For every x there is a point value of 1 for a o there is a point of -1. For every row and column we add the points. If there is a row or column with -3 or 3 points we know that there is where the player won. We also know based on if it is negative or positive if the winner was o or x. 

                        
    for (var row = 1; row <= 3; row++) {
